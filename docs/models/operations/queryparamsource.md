# QueryParamSource

`owned` — owner list (`userId` filter only).
`shared` — explicit share grant list (`isShared` + `sharedWith`).
Defaults to `owned` when omitted.

| Name                     | Value                    |
| ------------------------ | ------------------------ |
| `QueryParamSourceOwned`  | owned                    |
| `QueryParamSourceShared` | shared                   |
