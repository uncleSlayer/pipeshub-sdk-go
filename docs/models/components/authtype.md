# AuthType

Authentication type (required if connector supports multiple auth methods)


## Values

| Name                        | Value                       |
| --------------------------- | --------------------------- |
| `AuthTypeOauth`             | OAUTH                       |
| `AuthTypeOauthAdminConsent` | OAUTH_ADMIN_CONSENT         |
| `AuthTypeAPIToken`          | API_TOKEN                   |
| `AuthTypeUsernamePassword`  | USERNAME_PASSWORD           |
| `AuthTypeServiceAccount`    | SERVICE_ACCOUNT             |
