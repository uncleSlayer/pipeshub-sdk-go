# UserGroupType

Group type:
- admin: System admin group (cannot be modified)
- standard: Default user group
- everyone: All users group (cannot be modified)
- custom: User-created custom group



## Values

| Name                    | Value                   |
| ----------------------- | ----------------------- |
| `UserGroupTypeAdmin`    | admin                   |
| `UserGroupTypeStandard` | standard                |
| `UserGroupTypeEveryone` | everyone                |
| `UserGroupTypeCustom`   | custom                  |
