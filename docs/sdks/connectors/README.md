# Connectors

## Overview

### Available Operations

* [GetStats](#getstats) - Get connector statistics

## GetStats

Retrieve statistics for a specific connector including record counts, indexing status breakdown, and sync information.


### Example Usage

<!-- UsageSnippet language="go" operationID="getConnectorStats" method="get" path="/knowledgeBase/stats/{connectorId}" -->
```go
package main

import(
	"context"
	"os"
	"github.com/pipeshub-ai/pipeshub-sdk-go/models/components"
	pipeshub "github.com/pipeshub-ai/pipeshub-sdk-go"
	"log"
)

func main() {
    ctx := context.Background()

    s := pipeshub.New(
        pipeshub.WithSecurity(components.Security{
            BearerAuth: pipeshub.Pointer(os.Getenv("PIPESHUB_BEARER_AUTH")),
        }),
    )

    res, err := s.Connectors.GetStats(ctx, "<id>")
    if err != nil {
        log.Fatal(err)
    }
    if res.ConnectorStats != nil {
        // handle response
    }
}
```
