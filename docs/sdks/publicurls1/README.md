## SetFrontend

Configure the public URL where the frontend application is accessible. Used for OAuth redirects and email links.

### Example Usage

<!-- UsageSnippet language="go" operationID="setFrontendPublicUrl" method="post" path="/configurationManager/frontendPublicUrl" -->
```go
package main

import(
	"context"
	"os"
	"github.com/pipeshub-ai/pipeshub-sdk-go/models/components"
	pipeshub "github.com/pipeshub-ai/pipeshub-sdk-go"
	"log"
)

func main() {
    ctx := context.Background()

    s := pipeshub.New(
        pipeshub.WithSecurity(components.Security{
            BearerAuth: pipeshub.Pointer(os.Getenv("PIPESHUB_BEARER_AUTH")),
        }),
    )

    res, err := s.PublicUrls.SetFrontend(ctx, components.URLConfig{
        URL: "https://app.example.com",
    })
    if err != nil {
        log.Fatal(err)
    }
    if res != nil {
        // handle response
    }
}
```
