# AuthConfig

## Overview

### Available Operations

* [SetGoogle](#setgoogle) - Configure Google authentication

## SetGoogle

Set up Google OAuth as an authentication provider.

### Example Usage

<!-- UsageSnippet language="go" operationID="setGoogleAuthConfig" method="post" path="/configurationManager/authConfig/google" -->
```go
package main

import(
	"context"
	"os"
	"github.com/pipeshub-ai/pipeshub-sdk-go/models/components"
	pipeshub "github.com/pipeshub-ai/pipeshub-sdk-go"
	"log"
)

func main() {
    ctx := context.Background()

    s := pipeshub.New(
        pipeshub.WithSecurity(components.Security{
            BearerAuth: pipeshub.Pointer(os.Getenv("PIPESHUB_BEARER_AUTH")),
        }),
    )

    res, err := s.AuthConfig.SetGoogle(ctx, components.GoogleAuthConfig{
        ClientID: pipeshub.Pointer("123456789-abc.apps.googleusercontent.com"),
    })
    if err != nil {
        log.Fatal(err)
    }
    if res != nil {
        // handle response
    }
}
```
